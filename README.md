# zh-wdl-lab

本仓库用于归档、测试和发布多个独立的 HTML 页面。这些页面可能涉及不同主题与功能，彼此之间没有强关联，各自独立运行。仓库的主要目的是提供一个干净、可访问的静态页面存放空间，方便展示、分享或进一步开发。

## 项目简介

`zh-wdl-lab` 是一个轻量级的 HTML 页面集合仓库。所有页面均为纯前端实现，不依赖后端服务，可直接在浏览器中打开。仓库本身不针对任何特定域名或网站，仅作为页面归档与实验的场所。

## 目录说明

仓库根目录下直接存放各独立 HTML 文件，也可能包含少量配套资源（如 CSS、JavaScript、图片等），每个页面通常对应一个独立的文件夹或文件。目录结构示例：

```
/
├── index.html          # 仓库首页（可选）
├── page-a/
│   ├── index.html
│   └── assets/
├── page-b.html
└── README.md
```

- **根目录文件**：通常为仓库首页或说明文件。
- **子文件夹**：每个子文件夹代表一个完整的页面项目，包含其所有资源。
- **独立 HTML 文件**：可直接访问的单一页面，无额外资源依赖。

## 页面归档说明

本仓库中的页面均为历史或实验性内容，可能处于以下状态之一：

- **稳定版**：功能完整，可直接使用或参考。
- **开发中**：仍在迭代，可能存在未完成或临时内容。
- **归档版**：已停止维护，仅作为示例或记录保存。

每个页面内部可能包含简要说明，标注其状态与用途。建议在使用前查看页面内的注释或说明文档。
